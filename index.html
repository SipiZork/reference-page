Múlt

2004 óta foglalkozom programozással, fejlesztéssel. HTML-el kezdtem majd amikor ennek a korlátait elég hamar elértem, némi javascript és jQuery tudást szedtem magamra, a dinamikus tartalom megjelnítés jegyében.

Hamar elkérkezett a pont amikor valóban dinamikus tartalmakat szeretettem volna megjeleníteni, ekkor következett a PHP és a MySQL adatbázis.

Ezután több programozási nyelvbe is belekósoltam, mint C++, C#, Java. De velük csak egy-egy projekt erejéig foglalkoztam.

Sajnos az élet más felé szeretett volna terelni engem, és én engedtem neki.

Jelen

Egy nagyobb kihagyás után érett meg benne a gondolat, hogy nem szeretnék többet sodródni az árral. Szeretném én irányítani az életemet, és azzal foglalkozni ami valóban érdekel.

2020 elején elkezdtem, és egy hónapon belül be is fejeztem a Codeberry online kurzusát egészen a HTML alapoktól, a jQuery-n át, egészen egy alap React projektig. Nagyon megfogott, hogy a javascript meddig juttot, mi mindenre jó.

Így a React is nagyon meg tetszett, és a kurzus végeztél elkezdtem jobban beleásni magam a témába. Több udemy-s kurzust is végigvittem.

  • Complete React Developer in 2020
  • Build a Slack Chat App with React, Redux and Firebase

És több saját projektet is készítettem a megszerzett tudással, melyek a Referencia szekciónál megtekinthetőek.

Jövő

Szeretném megtalálni a helyemet egy remek csapatban. Az elején kezdeni, kérdezni, tanulni és fejlődni. Juniortól eljutni a Seniorig egy lépcsőt sem kihagyva, megdolgozva a sikerért.

Úgy felébredni minden reggel, hogy azzal foglalkozhatok amivel tényleg szeretek.
